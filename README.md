# CMAPSS - Prognostics Project

Proyecto base para pronóstico de vida útil remanente (RUL) con el dataset NASA CMAPSS.

## Estructura actual

- `data/`: datasets FD001-FD004 (train/test/RUL)
- `doc/`: documentos de referencia técnica
- `src/`: código fuente del pipeline (pendiente de implementación)
- `dashboard/`: app de visualización (pendiente de implementación)
- `fig/`: figuras y gráficos generados
- `out/`: salidas de entrenamiento/evaluación

## Plan de trabajo

1. Modelo de pronóstico
2. Agente
3. Dashboard

## Notas

- Este repositorio está en fase inicial.
- El dataset CMAPSS incluido es de texto plano y puede ocupar varios MB.
